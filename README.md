# Greenway Clinical Assist — PPC Landing Page Mock

Conversion-focused landing page for Greenway Health's Clinical Assist (AI medical scribe powered by Nabla), designed to capture paid search traffic on head terms like "ai medical scribe," "ambient ai documentation," and adjacent ambulatory-care commercial intent.

**Live preview:** https://ptolomea9.github.io/greenway-clinical-assist-mock/

## What this is

A single-file static HTML landing page built as a paid-traffic alternative to Greenway's existing product page at greenwayhealth.com/solutions/clinical-assist. The product page is editorial; this page is conversion-tuned.

## What's different from the product page

- **Head-term H1 match** — page title and H1 align directly to the keyword to lift Quality Score.
- **Inline lead form in the hero** — single conversion goal, no scroll required.
- **Inline ROI calculator** — interactive hours-saved + dollar-value model (no separate page).
- **Adjacent-themes bridge** — captures broader Novare commercial intent (prior auth, intelligent coding, patient portal, telehealth) without diluting the AI scribe focus.
- **Editorial visual system** — Gelasio display + Inter body + Greenway navy/teal, on an 8pt baseline grid with hairline-rule sectioning.
